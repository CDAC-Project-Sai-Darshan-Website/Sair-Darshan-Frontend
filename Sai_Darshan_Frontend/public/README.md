# Images Folder

## Required Images:

1. **sai-baba.jpg** - Add a Sai Baba image here
   - Recommended size: 200x200 pixels or larger
   - Format: JPG, PNG, or WebP
   - This image will be displayed in the login page header

## How to add the image:
1. Find a suitable Sai Baba image online
2. Save it as `sai-baba.jpg` in this public folder
3. The image will automatically appear in the login page

## Fallback:
If the image is not found, the component will show the prayer hands emoji (ğŸ™) as a fallback.
